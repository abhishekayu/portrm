# portrm

Blazing-fast, cross-platform CLI to inspect ports, kill processes, and recover broken dev environments.

## Install

```bash
pip install portrm
```

## Usage

```bash
# Scan all listening ports
ptrm scan

# Kill process on port 3000
ptrm kill 3000

# Inspect a port
ptrm info 8080

# Interactive TUI
ptrm interactive
```
